# EcoFly Dashboard
To run the dashboard locally you first need to create a python enviroment with these packages:

- pandas
- matplotlib
- seaborn
- openpyxl
- numpy
- shiny
- shinylive
- sympy

Then in the command line, run shiny with:   
`shiny run --port 39649 --reload --autoreload-port 46499 app.py`
